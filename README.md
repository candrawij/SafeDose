# Website Prediksi Dini Efek Samping Obat

## Deskripsi
Model machine learning dengan algoritma **Random Forest** dan **XGBoost** untuk preidksi dini efek samping obat.
Training model disimpan menggunakan format Pickle sehingga model tidak perlu training ulang setiap kali dijalankan
Penggunaan website menjadikan aplikasi ini lebih mudah diakses, dengan menggunakan framework **Streamlit** berbasis Python yang relatif mudah dan efisien.
